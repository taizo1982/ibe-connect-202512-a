### セクション3：Features

```
┌────────────────────────────────────────────┐
│  アプリでは分からなかった"人柄"に           │
│  出会える3つの理由                          │
│                                            │
│  ┌────────────────────────────────────┐    │
│  │ 👥 全員と1対1で話せるローテーション │    │
│  │ 「いいね」やプロフィールでは伝わらない │    │
│  │ 話し方・間合い・リアクションまで...   │    │
│  └────────────────────────────────────┘    │
│                                            │
│  ┌────────────────────────────────────┐    │
│  │ 🍷 3店舗はしごで自然に距離が縮まる  │    │
│  │ BAR → 居酒屋 → 焼肉と雰囲気が変わる │    │
│  └────────────────────────────────────┘    │
│                                            │
│  ┌────────────────────────────────────┐    │
│  │ 🔒 連絡先交換禁止。マッチングのみ   │    │
│  │ アプリのように"とりあえず"ではなく   │    │
│  └────────────────────────────────────┘    │
│                                            │
│  [[この仕組みで参加してみる]]               │
└────────────────────────────────────────────┘
```

**ポイント**:
- 3つの特徴すべてが「アプリとの違い」を強調
- ローテーション → 全員と話せる（アプリは選んだ人だけ）
- 3店舗 → 自然に距離が縮まる（アプリは会うまで分からない）
- 連絡先禁止 → 真剣な人だけ（アプリはとりあえず交換）
