## セクション3：Features（3つの特徴）

### 見出し
```
アプリでは分からなかった"人柄"に出会える3つの理由
```

### 特徴1
**タイトル**
```
少人数で、全員と1対1で話せるローテーション
```

**本文**
```
大人数の街コンのように「誰とも話せなかった」ということがないよう、
各店舗ではローテーション制で席替えを行い、
参加者全員と1対1でお話しできる時間をつくっています。

「いいね」やプロフィールでは伝わらない、
話し方・間合い・リアクションまで、じっくり感じることができます。
```

### 特徴2
**タイトル**
```
3店舗はしごで、自然に距離が縮まる
```

**本文**
```
最初は少し緊張していても、
BAR → 居酒屋 → 焼肉と雰囲気が変わるにつれて、
話題も空気も少しずつ砕けていきます。

「最初はぎこちなかったけれど、最後には自然に笑っていた」
そんな距離の縮まり方を、3店舗はしごの流れで設計しています。
```

### 特徴3
**タイトル**
```
連絡先交換は禁止。マッチングした人だけを公式LINEでサポート
```

**本文**
```
パーティー中の連絡先交換はあえて禁止にしています。

その代わり、
「また会って話したい」と思えたお相手の番号をカードに記入いただき、
マッチングが成立した方同士だけ、後日公式LINEからお見合い日程を調整。

アプリのように"とりあえず連絡を取る"のではなく、
お互いが「もう一度ちゃんと話したい」と思えた相手とのご縁だけをつなぎます。
```
